## 主要特性

### 设计风格
- **简洁优雅**：参考 Retrospect 主题的极简设计
- **摄影博客风格**：突出图片展示，适合摄影和生活类博客
- **左侧边栏布局**：固定侧边栏包含导航、关于信息和社交链接
- **大图展示**：文章题图自动提取并大尺寸展示

### 功能特性
- **自动题图提取**：从文章内容中自动提取第一张图片作为题图
- **响应式设计**：完美适配桌面、平板和移动设备
- **优雅交互**：平滑动画、悬停效果和加载状态
- **移动端友好**：侧边栏在移动端可收起，支持触摸操作

### 技术特性
- **模块化设计**：工具函数分离，代码结构清晰
- **性能优化**：图片懒加载、缓存机制
- **SEO 友好**：语义化 HTML、meta 标签优化
- **可维护性**：模板系统、样式组织良好

## 使用说明

### 启动开发服务器
```bash
npm run dev
```

### 访问地址
- 前台：http://localhost:8787
- 后台：http://localhost:8787/admin

### 注意事项
1. 需要 Node.js v20.0.0 或更高版本
2. 文章题图会自动从内容中的第一张图片提取
3. 如果文章没有图片，则不显示题图
4. 移动端通过左上角菜单按钮切换侧边栏
