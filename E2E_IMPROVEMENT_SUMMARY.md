## 发现的问题

### 服务器端问题

测试过程中发现了一些服务器端问题，但这些不影响测试通过：

1. **D1 数据库错误**：首页加载时出现 D1_ERROR，与 Zod 解析相关
   ```
   Error: D1_ERROR: Failed to parse body as JSON, got: ZodError
   ```

2. **仪表盘错误**：访问 `/admin/dashboard` 时出现空指针异常
   ```
   TypeError: Cannot read properties of null (reading 'display_name')
   ```

这些问题不影响 E2E 测试的通过，因为测试只验证页面是否可访问。

## 使用说明

### 运行所有 E2E 测试

```bash
export PATH="/usr/local/opt/node@20/bin:$PATH"
npm run test:e2e
```

### 运行特定浏览器测试

```bash
npm run test:e2e:chromium   # 仅 Chrome
npm run test:e2e:firefox     # 仅 Firefox
npm run test:e2e:webkit      # 仅 Safari
```

### 调试模式

```bash
npm run test:e2e:debug
```

### 查看测试报告

```bash
npm run test:e2e:report
```

## 注意事项

1. **Node.js 版本**：确保使用 Node.js v20 或更高版本
2. **环境变量**：`start-dev-server.sh` 已自动设置 PATH
3. **服务器自动启动**：Playwright 会自动启动开发服务器，无需手动启动
4. **测试隔离**：每个测试之间没有状态依赖
